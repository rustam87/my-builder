# Project Title

My builder

## Description
Проект тестового задания

## Instruction

1. Перейдите в директорию проекта
```
cd projectName
```
2. Далее ввести команду для установки зависимотсей
```
npm i
```
3.  Установить gulp (в нашеи случае gulp 4)
```
npm install gulpjs/gulp.git#4.0 --save-dev
```
4. Запустить gulp
```
gulp
```
